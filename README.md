### Quick Start
1.  `git clone <repo>`
2.  `npm install` in both `frontend/` and `backend/`.
3.  Set `.env` variables (see sub-READMEs).
4.  Run:
    *   Backend: `npm start`
    *   Frontend: `npm run dev`
